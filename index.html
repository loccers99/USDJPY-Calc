Market & Account

If your broker pays/charges $X per $1 move per 1 lot, set Point Value = X (default 1).

Risk & Bands
Bands (in code): 1–3 → 0.5%, 4–7 → 1.0%, 8–10 → 1.25%, ≥11 → 1.5%.
Stop & Targets (ATR multiples)

This calculator uses a static stop distance (SL × ATR) for position sizing. No trailing/offsets are used.

Execution Mode

Partial TP: calculator shows Standard SL, TP1, and TP2. Sizing uses static SL × ATR.

Single: calculator shows Standard SL and Standard TP (from Single TP Multiplier). Sizing uses static SL × ATR.
